# TriClapClap
Легковесный оверлей нажатий клавиш для OBS Studio

## Особенности
* **Раздельный звук:** Независимое управление звуком локально (на ПК) и на стриме (через WebSocket) позволяет оставить звук только там, где необходимо.
* **Горячая перезагрузка:** Обновление конфигурации (`config.json`) и папки с файлами (`Assets/`) на лету без перезапуска приложения.
* **WebSocket Сервер:** Передача кадров анимации, общего счетчика и CPS в реальном времени для интеграции с браузерным оверлеем.

## Инструкция по использованию

### Способ 1: Интеграция через браузер (Рекомендуемый)
*Обеспечивает чистую прозрачность для ассетов персонажа без артефактов хромакея.*

1. Запустите `triclapclap.exe`.
2. Добавьте источник **Браузер (Browser Source)** в OBS.
3. Отметьте галочку **Локальный файл** и выберите `index.html`.
4. Установите разрешение **Ширина: 1024**, **Высота: 576** *(можно изменить внутри HTML-файла)*.
5. Если включен звук для стрима, отметьте флаг **Управлять аудио с помощью OBS** в свойствах источника.

> 📌 **Примечание:** Чтобы звук был слышен только на стриме, установите параметры `"IsSoundLocalEnabled": false` and `"IsSoundStreamEnabled": true` в файле `config.json`.

### Способ 2: Захват окна (Альтернативный)
1. Запустите `triclapclap.exe`.
2. Добавьте источник **Захват окна** в OBS.
3. Выберите окно `triclapclap` из списка.
4. Нажмите правой кнопкой мыши по источнику -> **Фильтры**.
5. Добавьте фильтр **Хромакей** для удаления цвета заднего фона, указанного в конфигурации.

## Сборка приложения

### Способ 1: Через Visual Studio
1. Откройте файл проекта (`.csproj`) или решение (`.sln`) в Visual Studio.
2. Установите конфигурацию сборки на **Release** на верхней панели инструментов.
3. Соберите решение, нажав **`Ctrl + Shift + B`** (или через *Сборка -> Собрать решение*).

### Способ 2: Через .NET CLI
