# triclapclap
Легковесный графический оверлей нажатий клавиш для OBS Studio.

## Возможности
* **Hot-Reload:** динамическая загрузка конфигурации (`config.json`) и внешних ассетов (`Assets/`) без перезапуска приложения.
* **WebSocket Сервер:** асинхронная трансляция кадров анимации и счетчиков для интеграции через Browser Source.

---

## Способ 1: Интеграция через веб-интерфейс (Рекомендуемый)
*Обеспечивает чистую альфа-прозрачность персонажа без рваных пикселей по краям.*

1. Запустите `triclapclap.exe`.
2. Добавьте в OBS источник **«Браузер» (Browser Source)**.
3. Включите чекбокс **«Локальный файл»** и укажите путь к файлу `index.html`.
4. Установите разрешение: **Ширина: 1024**, **Высота: 576** *(изменяется в HTML при необходимости)*.

> 📌 **Примечание:** Файл `index.html` предоставлен в качестве базового шаблона. Вы можете изменять его CSS-структуру и логику под свои задачи.

---

## Способ 2: Прямой захват окна (Альтернативный)
1. Запустите `triclapclap.exe`.
2. Добавьте в OBS источник **«Захват окна» (Window Capture)**.
3. Выберите окно приложения `triclapclap`.
4. Нажмите ПКМ по созданному источнику -> **«Фильтры» (Filters)**.
5. Добавьте фильтр **«Хромакей» (Chroma Key)** для удаления фонового цвета, заданного в конфигурации.

## Сборка приложения

### Способ 1: Через Visual Studio
1. Откройте проект (`.csproj`) или файл решения (`.sln`) в Visual Studio.
2. Переключите конфигурацию сборки на **Release** на верхней панели инструментов.
3. Выполните сборку комбинацией клавиш **`Ctrl + Shift + B`** (или *Сборка -> Собрать решение*).

### Способ 2: Через командную строку (.NET CLI)
Выполните команду в корневой папке проекта (где находится `.csproj`):
```bash
dotnet build -c Release
```
